Introduction

Vous avez une entreprise qui vend des produits ou offre des services. Vous avez indispensablement besoin d'une page Facebook pour entreprise pour réussir votre Marketing Digital.

L'utillisation mensuelle de Facebook est estimée à 1.96 milliards d'utillisateurs. A travers la page Facebook de votre entreprise, vous pouvez faire la pubicité de vos produits ou services à des milliers de clients.

A travers votre Page Enreprise, Facebook vous donne accès à des outils de marketing très puissants comme la publication du contenu, des vidéos directes, des publicités, etc.

Dans cet article, vous allez découvrir comment créer une page Facebook pour votre entreprise.

Avant de commencer

Pour créer une Page Facebook pour Entreprise, vous devez avoir un compte Facebook. Pour créer un Compte Facebook, rendez vous sur la page facebook.comet suivez les instructions.

Une fois votre Compte Facebook créé, connectez vous à votre Compte et vous êtes maintenant prêts à créer votre Page Facebook pour Entreprise.

Comment créer votre page Facebook

Etape 1: Icône du Menu

Lorque vous êtes connectés à votre compte, cliquez sur l'icône du menu en haut à droite de la page puis cliquez sur le lien "Pages".

Etape 2: Création de la Page

Remplissez les trois (3) champs qui sont:

  • •  Nom de votre page: Choisissez un nom facile à retenir et si possible qui s'identifie au mieux à votre domaine;
  • •  Catégorie de votre page: Choisissez trois (3) catégories qui décrivent au mieux votre entreprise;
  • •  Description de votre page: Donnez une description détaillée de vos services ou produits.

Puis, cliquez sur le button "Créer une Page".

Etape 3: Ajouter des Photos

Vous êtes maintenant sur votre nouvelle page. Continuer la mise en place de la page de votre entreprise en ajoutant une photo de couverture et une photo de profil.

  • •  Photo de Couverture: De taille 820px largeur 312px hauteur, doit être une image originale montrant vos produits, vos services ou votre local;
  • •  Photo de profil: De taille 170px/170px, doit être idéalement le logo de votre entreprise.

Etape 4: Ajouter un nom d'utilisateur pour la Page

Facebook vous donne la possibilité d'avoir un lien personnalisé vers votre Page. En créant un nom d'utilisateur unique (ex: nom_entreprise), les visiteus pourront acceder à votre Page via le lien www.facebook.com/nom_entreprise

Idéalement, le nom d'utilisateur de votre Page doit être le nom de votre entreprise. Si ce nom n'est pas disponible vous pouvez ajouter un mot comme _officel

Votre nom d'utilisateur ne doit pas dépasser 50 caractèrs. Eviter un nom d'utilisateur inutilement long.

Pour créer un nom d'utilisateur, cliquer sur Créer un @nomutilisateur. Essayer d'obtenir une coche verte avant de valider.

Etape 5: Ajouter un Bouton d'Appel à l'Action

Le bouton Appel à l'Action permet à vos visiteurs de réaliser une action rapide qui est importante pour votre entreprise. Cela peut être une visite de votre site web, votre contact Whatsapp, contact par téléphone, contact par email, etc.

Pour ajouter un bouton d'Appel à l'Action, cliquer sur le bouton Ajouter un bouton puis choisir le bouton à ajouter

Etape 6: Compléter les Informations de votre Page

Une Page Facebook de votre entreprise vous permet d'ajouter pratiqement toutes les informations sur votre entreprise. Votre Page Facebook offre aux visiteurs une sorte une visite virtuelle de votre entreprise.

Pour tirer l'avantage de cela, vous devez compléter les informations de votre Page.

Pour compléter votre Page, défiler un peu vers le bas et cliquer sur le bouton Entrer les informations et les préférences puis compléter les champs.

Etape 7: Inviter des amis à aimer votre Page

Maintenant que votre Page est créée, vous pouvez commencer à gagner des audiences en invitant vos amis sur Facebook à aimer votre Page.

Cette première audience est gagnée presque sans effort donc il faut en profiter.

Pour inviter des amis à aimer votre Page, défiler encore un peu vers le bas et cliquer sur le bouton Présenter votre Page puis commencer à inviter.

Promouvoir votre page Facebook en étant actif

La création d'une Page Facebook constitue la première étpe de votre Marketing sur Facebook.

Pour retenir votre audience et en gagner d'autre, votre Page d'être active. Pour cela il faudra constamment créer des Publications, ajouter des Stories et faire des Direct

Vous pouvez augmenter la visite de votre Page Facebook en ajoutant le lien vers votre Page dans votre site web, votre blog et votre Page sur d'autre résau social comme LinkedIn

Conclusion

L'utillisation des Médias Sociaux notamment Facebook est incontournable dans votre stratégie de Marketing Digital.

Vous pouvez utiliser la Page Facebook de votre entreprise pour non seulement convertir votre audience en des clients de votre entreprise, mais aussi pour en trouver d'autres.

Votre Page Facebook vous donne accès à des milliers de clients potentiels rien que dans votre emplacement.
